# Lead Sniper Assignment (n8n)

## 🚀 Overview

This project is an automated workflow built using n8n to identify high-value GitHub users and send AI-generated insights to Discord.

## ⚙️ Workflow Steps

1. Fetch GitHub user data using GitHub API
2. Filter users:

   * Followers > 100 OR
   * Public repos > 50
3. Generate AI-based sales pitch
4. Send result to Discord

## 🧠 Features

* AI-powered automation
* Lead filtering logic
* Discord webhook integration
* Handles missing bio
